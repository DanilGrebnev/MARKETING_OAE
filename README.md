# Neon Insight Lab Landing

## Formspree
1. Создайте форму в Formspree и получите ID.
2. Замените `YOUR_ID` в атрибуте `action` формы (`index.html`) и, при необходимости, в `form.js` константе `endpoint`.
3. В панели Formspree отметьте разрешённые домены и включите уведомления.

## Netlify Forms
- Поля `data-netlify` и `netlify-honeypot` уже добавлены. Задеплойте проект на Netlify — форма автоматически появится в разделе Forms.
- Для теста из Netlify CLI используйте: `netlify deploy --prod`.
- При необходимости добавьте скрытое поле `form-name` (уже есть) с таким же значением в панели.

## Деплой
- Любой static-хостинг: загрузите файлы целиком (`index.html`, `style.css`, `form.js`, ассеты).
- Для локальной проверки откройте `index.html` в браузере или используйте `npx serve .`.
- При использовании CI/CD убедитесь, что загружается вся папка `/marketing`.

## Макеты (Figma / Sketch frames)
- Desktop (1440?900): сетка 12 колонок, hero в две колонки, сфера справа, карточки по 4 в ряд.
- Tablet (992?768): hero в одну колонку, сфера над текстом, карточки 2?2, форма на всю ширину.
- Mobile (390?844): hero-типографика + статичное SVG, карточки стеком, кнопки формы на всю ширину.
